isdigit(char) => returns true if the passed char is number

isalpha(char) => returns true if the passed char is letter (upper case or lower case)

isalnum(char) => returns true if the passed char either letter or number

isupper(char) => returns true if the passed char is upper case (capital)

islower(char) => returns true if the passed char is lower case (small letter)

toupper(char) => returns char in upper case

tolower(char) => returns char in lower case

reverse(s2.begin(), s2.end());  => reverses any container

sort(s.rbegin(), s.rend()); => sort from bigger to smaller

sort(s.begin(), s.end()); => sort from smaller to bigger

sort (array, array + size); => sort the array

getline(cin , string name); => cin full string with spaces

cin.ignore(1,'\n'); >= to do 2 or more getlines
